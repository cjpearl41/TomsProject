Frequently Asked Questions

Q: Is the price on the website the set price?

A: No, the price on the website is a combination of the average price, and an estimate. The price can fluctuate based on the size of the project and length of the project.

Q:How long does it take to complete each project?

A: The length of each project depends on the size of it, if it is a building extension, it will take more time, but if it is a portable delivery it won't take long. But whatever it is we do, we try to do it in a timely manner.
